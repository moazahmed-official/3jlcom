# Reports

- POST /reports — Create a report (ad/user/dealer). Schema: `ReportCreate`.
- GET /reports — List reports (admin/moderator).
- GET /reports/{reportId} — Get report details.
- DELETE /reports/{reportId} — Delete report (admin).

Schemas: `ReportCreate`, `ReportResponse` in [api-catalog/openapi.bundle.yaml](api-catalog/openapi.bundle.yaml).
