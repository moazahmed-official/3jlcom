# API documentation (Markdown)

These Markdown files are a human-friendly representation of the OpenAPI spec. They are derived from `openapi.bundle.yaml` and intended for quick reading or inclusion in documentation sites.

Files in this folder:

- `auth.md` — Authentication endpoints
- `users.md` — User management
- `normal-ads.md` — Normal ads CRUD
- `unique-ads.md` — Unique ads CRUD
- `caishha.md` — Caishha ads and offers
- `auctions.md` — Auctions and bids
- `media.md` — Media upload and retrieval
- `offers.md` — Generic offers
- `brands-models.md` — Brands & models
- `reviews.md` — Reviews
- `reports.md` — Reports
- `packages.md` — Packages and assignments
- `notifications.md` — Notifications
- `sliders.md` — Frontend sliders
- `findit.md` — Findit ad type (requests & offers)

Full machine-readable spec is available at: [api-catalog/openapi.bundle.yaml](api-catalog/openapi.bundle.yaml)
